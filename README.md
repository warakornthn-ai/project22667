# 🐾 Pet Clinic Management System

## 📌 รายละเอียดโปรเจค

ระบบจัดการคลินิกสัตว์เลี้ยง (Pet Clinic Management System) ถูกพัฒนาด้วย Django
เพื่อช่วยบริหารจัดการข้อมูลภายในคลินิก เช่น ข้อมูลลูกค้า สัตว์เลี้ยง การนัดหมาย และการรักษา

---

## 🎯 วัตถุประสงค์

* เพื่อพัฒนาระบบช่วยจัดการข้อมูลภายในคลินิกสัตว์เลี้ยง
* เพื่อลดความผิดพลาดในการจัดเก็บข้อมูล
* เพื่อเพิ่มความสะดวกในการใช้งานสำหรับเจ้าหน้าที่

---

## ⚙️ ฟีเจอร์หลักของระบบ

* 📋 จัดการข้อมูลลูกค้า (Customer Management)
* 🐶 จัดการข้อมูลสัตว์เลี้ยง (Pet Management)
* 📅 ระบบนัดหมาย (Appointment System)
* 💊 ระบบการรักษาและใบสั่งยา (Treatment & Prescription)
* 🧾 ระบบใบเสร็จ (Billing / Receipt)

---

## 🛠️ เทคโนโลยีที่ใช้

* Python
* Django Framework
* SQLite (หรือฐานข้อมูลที่ใช้)
* HTML / CSS / Bootstrap

---

## 🚀 วิธีติดตั้งและใช้งาน

### 1. ติดตั้ง Django

pip install django

### 2. เข้าไปในโฟลเดอร์โปรเจค

cd vetclinic

### 3. รันเซิร์ฟเวอร์

python manage.py runserver

### 4. เปิดเว็บ

เข้าใช้งานผ่านเบราว์เซอร์:

http://127.0.0.1:8000/

---

## 📂 โครงสร้างโปรเจค (ตัวอย่าง)

* models.py → โครงสร้างฐานข้อมูล
* views.py → การทำงานของระบบ
* templates/ → หน้าเว็บ
* static/ → ไฟล์ CSS / JS
